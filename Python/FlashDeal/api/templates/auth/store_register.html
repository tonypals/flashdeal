Vad vi lagrar om er butik
FlashDeal lagrar de uppgifter ni anger vid registrering: företagsnamn, kontaktperson, e-postadress, telefon och adress. Dessa används för att:
  • Visa er butik och era erbjudanden på plattformen
  • Kontakta er om bokningar, godkännanden och tekniska frågor
  • Hantera utbetalningar via Stripe Connect
Hur länge
Uppgifterna lagras så länge ert konto är aktivt. Ni kan begära radering av konto och data via e-post till gdpr@flashdeal.se.

Delning med tredje part
Vi delar aldrig era uppgifter med annonsörer. Stripe får de uppgifter som krävs för betalningshantering. Vi använder Supabase som databasplattform inom EU.

Personuppgiftsansvarig är FlashDeal Sverige AB. Ni har rätt att begära utdrag, rättelse och radering av era uppgifter enligt GDPR.
